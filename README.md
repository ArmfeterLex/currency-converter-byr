# Currency Converter BYR

Модуль для консольных приложений на C#, конвертирующий стоимость товара из белорусских рублей в российские рубли, евро и гривны по фиксированным курсам.

## Функциональность

- Ввод стоимости товара в белорусских рублях (целое число)
- Конвертация по фиксированным курсам:
  - Белорусский рубль → Российский рубль: × 28.8
  - Белорусский рубль → Евро: × 0.2737
  - Белорусский рубль → Гривна: × 12.6
- Округление результатов до двух знаков
- Вывод всех трёх значений

## Пример работы

Введите стоимость товара в белорусских рублях
100
Стоимость товара в русских рублях 2880.00; в евро 27.37; в гривнах 1260.00

Введите стоимость товара в белорусских рублях
50
Стоимость товара в русских рублях 1440.00; в евро 13.69; в гривнах 630.00

## Требования

- .NET Core / .NET Framework
- Консольное приложение

## Примечание

Курсы валют зафиксированы в коде (28.8, 0.2737, 12.6). Для актуальных курсов требуется обновление значений или подключение API.
